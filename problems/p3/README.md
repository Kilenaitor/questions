# Array Combo
## Instructions
1. Given two sorted arrays, A and B, create a method that returns a single array made up of alternating elements from each array.
2. Output the resulting array to standard out.
## Example
> A = [1, 2, 3, 4, 5]
> B = [A, B, C, D, E]
> output : [1, A, 2, B, 3, C, 4, D, 5, E]
